Cài đặt AWS CLI và Docker Desktop

2.1 Cài đặt AWS CLI

2.1.1 Yêu cầu tối thiểu

  • Hệ điều hành: Windows, macOS hoặc Linux.
  • Đã có tài khoản AWS.

2.1.2 Các bước

  1. Tải AWS CLI

    • Windows: Tải file cài đặt từ AWS CLI v2 Download.
    • macOS: Cài qua brew hoặc tải file .pkg.
    • Linux: Sử dụng gói .zip hoặc qua trình quản lý gói (apt, yum).

  2. Cài đặt

    • Windows: Mở file .msi và làm theo hướng dẫn cài đặt.
    • macOS: Mở file .pkg hoặc chạy: 
      brew install awscli
    • Linux: 
      curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

  3. Kiểm tra phiên bản

    aws --version

  4. Cấu hình AWS CLI

    aws configure

  5. AWS CLI sẽ yêu cầu nhập thông tin

    AWS Access Key ID [None]: <your key>
AWS Secret Access Key [None]: <your secret key>
Default region name [None]: <your region>
Default output format [None]: <press Enter>

2.2 Cài đặt Docker Desktop

Cài đặt Docker Desktop

2.2.1 Yêu cầu tối thiểu

  1. Windows
    • Windows 10/11 64-bit.
    • Enable WSL 2 hoặc Hyper-V.
    • Tối thiểu 4GB RAM.
  2. macOS
    • macOS 11 (Big Sur) trở lên.
    • Hỗ trợ chip Intel or Apple Silicon (M1/M2) chips.

2.2.2 các bước

  • Truy cập: https://www.docker.com/products/docker-desktop
  • Chọn phiên bản hợp với hệ điều hành:
    • Windows (x86_64 / ARM64)
    • Mac (Intel / Apple Silicon)
    • Linux

  1. Cài đặt

    1.1 Windows

    • Mở file .exe vừa tải.
    • Chọn Use WSL 2 instead of Hyper-V (khuyến nghị).
    • Nhấn Next và hoàn tất cài đặt.
    • Khởi động lại máy nếu cần.

    1.2 macOS

    • Mở file .dmg vừa tải.
    • Kéo Docker.app vào thư mục Applications.
    • Mở Docker và cấp quyền chạy.

  2. Xác minh và cài đặt

Mở Terminal hoặc CMD và chạy lệnh:

docker --version
docker compose version
